# Opposite face of a dice

## Description
An easy and short java program that gives the opposite side of a dice

## Features 
- Takes user input
- Displays output on console

## Technologies used 
-java

## How to Run
1. Compile the program using 'javac Dice'
2. Run the program using 'java Dice'

## Concepts used
- Simple mathematics

## Output
displays the number that at the opposite face of the given face number

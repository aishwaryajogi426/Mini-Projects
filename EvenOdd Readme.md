# Even Odd Checker

## Description
A simple Java program that checks whether a number is even or odd.

## Features
- Takes user input
- Displays result on console

## Technologies Used
- Java

## How to Run
1. Compile the program using `javac EvenOdd.java`
2. Run the program using `java EvenOdd`

## Concepts Used
- If-else
- Bitwise operators

## Output
Displays whether the entered number is even or odd.

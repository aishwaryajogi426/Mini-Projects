# Multiplication table

## Description
A simple java program which prints the multiplication table upto 10x of a number

## Features
- Takes user input
- Displays output on console

## Technologies used
-Java

## How to Run
1. Compile the program using 'javac Table.java'
2. Run the program using 'java Table'

## Concepts used
- For loop

## Output
Displays the multiplication table of a number upto 10

# Even Odd Checker

## Description
A simple Java program that gives the sum of the first n numbers.

## Features
- Takes user input
- Displays result on console

## Technologies Used
- Java

## How to Run
1. Compile the program using `javac SumN.java`
2. Run the program using `java SumN`

## Concepts Used
- For loop

## Output
Displays the sum of the numbers from 1 to the given number

# nth term of an arithmetic progresssion

## Description
A simple Java program that finds the nth term of an arithmetic progression.

## Features
- Takes user input of first and second term.
- Displays result on console

## Technologies Used
- Java

## How to Run
1. Compile the program using `javac Nterm.java`
2. Run the program using `java Nterm`

## Concepts Used
- logic
- Basic maths

## Output
Displays the nth term of an AP
